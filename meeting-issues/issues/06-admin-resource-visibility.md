## Problem
No simple way for admins to choose which resources feed the RAG and in what priority.

## Proposed solution
A right-sidebar UI to select/deselect and prioritize resources, with a clear taxonomy (Bibles, study notes, translation notes, etc.) showing which MCP server each came from. Likely drag-and-drop prioritization rather than a hard all-vs-curated split.

## Scope
- [ ] Design the sidebar (select + drag-and-drop ordering)
- [ ] Define taxonomy + per-MCP-server attribution
- [ ] Depends on MCP resource enumeration
